## 2. 밴픽 화면 레이아웃

### 2.1 상단 바 (라운드/타이머/팀 정보)

- 구성
  - 좌측: Player A 이름, 종족 아이콘, 색상, 시리즈 스코어.
  - 중앙:  
    - 현재 단계 텍스트: `BAN 1/2`, `PICK 2/3` 등.  
    - 큰 타이머(초 단위) + 얇은 진행 바.
  - 우측: Player B 동일 구조.
- 스타일
  - LoL 챔프 선택 상단 구조를 참고, 숫자와 라벨을 크게 배치.[web:209][web:232]
  - GameHeart 상단 플레이어 정보와 톤을 맞춰서, 인게임 HUD와의 연속성 확보.[web:225][web:230]

---

### 2.2 중앙 유닛 그리드 (A안 베이스)

- 역할
  - 밴/픽 후보 전체를 한 눈에 보여주는 **메인 탐색 영역**.
- 구성
  - 6×N ~ 8×N 아이콘 그리드:
    - 각 아이콘: 유닛 초상화, 테두리(종족 색 또는 역할 색), 밴 여부 오버레이.[web:209]
  - 상단에 필터/검색 바:
    - 종족, 역할(탱커/딜러/지원 등), 티어, 텍스트 검색.[web:220]
- 동작
  - 클릭 시 해당 유닛이 하단/우측 미리보기 영역에 크게 표시.
  - 이미 밴/픽된 유닛은 회색 처리 + 클릭 불가.[web:209][web:232]

---

### 2.3 좌/우 밴 리스트 (A안 요소)

- 좌측 세로 스택: **내 팀이 한 밴/자기밴 리스트**.
- 우측 세로 스택: **상대에게 들어간 밴 리스트**.
- 구분 방식
  - 공개 밴: 선명한 아이콘 + 팀 컬러 테두리.
  - 쉐도우 밴: 실루엣/물음표 아이콘 → 게임 시작 시 실제 아이콘으로 바뀜.
  - 자기밴: 유닛 아이콘 + 붉은 X 오버레이.
- 참고
  - LoL/대회용 픽밴 오버레이의 슬롯 배열 구조 활용.[web:216][web:224]

---

### 2.4 하단 정보/확인 영역 (A안 + B안 일부)

- 좌측
  - 선택된 유닛 미리보기:
    - 큰 초상화, 이름, 종족/역할 아이콘, 한 줄 요약.[web:209][web:220]
- 중앙
  - “선택/밴 확정” 버튼, “랜덤” 버튼(옵션).
- 우측 (B안 요소)
  - 간단 추천:
    - “현재 조합에서 부족한 역할” 아이콘 1~2개.
    - “이 유닛은 vs Zerg에 강함” 같은 짧은 텍스트 1줄.[web:220][web:234]
- 참고
  - 도타2의 “역할/난이도/추천” 박스를 축소해서 옮긴 느낌으로 구축.[web:220][web:214]

## 3. 역할/추천 영역 (B안 핵심 요소 축소 버전)

### 3.1 역할 필터/태그

- 중앙 그리드 상단에 “역할 필터 바” 추가:
  - 예: `탱커 / 근접 딜러 / 원거리 딜러 / 캐스터 / 지원 / 공성`.
- 각 유닛 메타 데이터에 복수 역할 태그를 달아서,  
  도타2 히어로 필터처럼 동작하게 설계.[web:220][web:228]

### 3.2 간단 추천 로직 (1차 버전)

- 규칙 기반(데이터만으로 구현 가능한 수준):
  - 상대 종족/상대가 밴한 유닛에 따라:
    - “vs Mech에 강한 유닛”, “공성 부족해 보입니다” 정도의 힌트.[web:234]
- UI
  - 하단 우측에 **작은 텍스트/아이콘 2~3개**만 배치해서,  
    정보량은 늘리되 화면을 복잡하게 만들지 않기.

## 4. 인게임 HUD와의 연결

### 4.1 GameHeart 상단 HUD 확장

- 목표
  - 밴픽에서 본 정보와 인게임 HUD가 **시각적으로 이어지도록** 하는 것.
- 방식
  - GameHeart 상단 플레이어 이름 아래에 “밴 바” 추가:
    - A/B 팀 각각 한 줄, 작은 아이콘 3~5개.[web:225][web:230]
    - 공개 밴/자기밴만 실제 아이콘, 쉐도우 밴은 실루엣 또는 빈 슬롯.
- 스타일
  - 런처 밴픽 슬롯과 동일한 아이콘/테두리 스타일 사용:
    - 사용자/관전자 입장에서 자연스럽게 연결됨.[web:209][web:225]

### 4.2 관전자용 추가 정보 (B안 요소)

- 옵저버 모드에서만:
  - 밴 아이콘 위에 마우스를 올리면,
    - 도타2처럼 **역할/주요 특징**이 툴팁으로 뜨게 할 수 있음.[web:220][web:234]
- 방송용
  - 필요하면, `lol-pick-ban-ui`처럼 별도 대회 오버레이에  
    밴/픽 정보를 한 번 더 정리해주는 것도 확장 포인트.[web:216]

## 5. 색상/폰트 가이드 (초안)

- 배경
  - 아주 어두운 남색/메탈릭 그레이 (SC2 기본 톤과 맞춤).[web:233]
- 팀 컬러
  - Player A: 청색 계열, Player B: 주황/빨강 계열 (GameHeart 관례).[web:225][web:230]
- 밴 타입
  - 공개 밴: 팀 컬러 테두리.
  - 쉐도우 밴: 회색/보라 테두리 + 반투명 아이콘.
  - 자기밴: 붉은 X 오버레이.
- 폰트
  - 타이머/라운드/플레이어 이름은 크게,  
    설명/추천 텍스트는 작은 두께 얇은 폰트.
